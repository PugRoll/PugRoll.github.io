Introduction:

This was a quick and dirty project I threw together to grasp java’s awt class. As I had not properly been introduced to how it works.

Requirements:

  1. This project should be contained in one class file in Java
    1. I didn’t want to overcomplicate what should be a simple project
  2. Use the AWT to create a small window to handle the the buttons and labels.
    Otherwise I wasn’t too concerned for background colors or added functionality.
